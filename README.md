## 项目描述

使用原生 PHP 对数据库进行操作，并将数据渲染到前端页面

功能：

- 商品添加
- 商品修改
- 商品删除
- 模糊搜索
- 商品状态为“已上架”不能修改及删除

## 项目基本信息

Desktop System：macOS

IDE ：PhpStorm 2021.3

PHP：7.4.21

MySQL：5.7.34

## 项目结构

```
./mall2.0  // 项目根目录
  ├── README.md
  ├── create.php // 添加逻辑处理
  ├── delete.php // 删除逻辑处理
  ├── mall.php // 主程序（页面）
  └── mall.sql // 数据库脚本
```
